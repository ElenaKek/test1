Единицы измерения цветов

Существует множество вариантов определения цвета в CSS, некоторые из которых используются довольно часто по сравнению с другими. Одинаковые значения цвета Организации Объединенных Наций в CSS определяют ли вы при этом цвет текста, цвет фона или цвет чего-либо ещё.

Стандартная система цветов, доступная в современных компьютерах, — это 24-битная система, которая позволяет отображать около 16,7 миллионов редких цветов благодаря использованию различных красных, зелёных и синих трактов с 256 значениями каждого канала (256 x 256 x 256 = 16 777 216). Давайте взглянем на некоторые, с помощью которых мы используем определение цвета в CSS.

Cпособы обозначения цвета

  1. Ключевые слова цвета

  2. Шестнадцатеричные значения RGB

    Случайное шестнадцатеричное значение состоит из символа решётки (#) за элементами 6 шестнадцатеричных чисел, каждое из которых может принимать одно из 16 значений от 0 до f (представляющая 15) — то есть 0123456789abcdef. Каждая пара значений представляет собой одно из проявлений — красного, зеленого или синего цветов — и позволяет нам определить любые из 256 доступных значений для каждого (16 x 16 = 256).

    3. Эти значения хотят быть более универсальными, чем ключевые слова — вы можете использовать шестнадцатеричное значение, чтобы отобразить любой цвет, который вы используете в своей формулировке цветов.

  3. RGB и RGBA значения

    Значения RGB это функция — rgb()которая даёт три параметра отображающих красного, зеленого и синего значений цветов, во многом так же, как шестнадцатеричное значение. Отличие от RGB заключается в том, что каждый канал представлен не двумя шестнадцатеричными цифрами, а десятичным числом между 0 и 255 — что отчасти проще в восприятии.
